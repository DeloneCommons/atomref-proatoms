# AI assistance note

scientific discussion, implementation, review, and documentation drafting.

The repository author is responsible for the final scientific content, code,
data curation, validation criteria, documentation, licensing choices, and release
decisions. AI-generated material is not an authoritative source, a validation
result, or a coauthor.

frozen basis bundles, workflow scripts, local SCF artifacts used during release
generation, and generated QA reports. Those artifacts, not the AI assistance
process, define the scientific and technical content of the release.
