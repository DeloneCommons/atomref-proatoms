# Data layout

The initial repository data layer is copied from `atomref-proatoms-data-v2.zip`.

```text
data/states/       source / selection / curated atomic states
data/basis_sets/   frozen BSE NWChem spherical basis bundles
data/profiles/     future generated radial-density datasets
```

`local-data/` is ignored and is reserved for local SCF logs, checkpoints, scratch files,
and temporary wavefunction artifacts.
