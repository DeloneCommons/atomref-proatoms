# Example 01: neutral Multiwfn `.rad` and `.wfn` files from BSE

This beginner example generates neutral atom references for:

```text
H and B--F
```

It demonstrates:

- explicit element selection with `--elements H`;
- range selection with `--element-range B-F`;
- use of a Basis Set Exchange basis with `--basis bse:cc-pVDZ`;
- non-relativistic generation with `--relativity none`;
- Multiwfn-oriented outputs with `--artifacts rad,wfn`.

Run:

```bash
./run.sh
```

The output committed in `output/` was generated by this script. The important
files are:

```text
output/multiwfn/rad/   # density-only Multiwfn .rad files
output/multiwfn/wfn/   # neutral PROAIM .wfn files
output/multiwfn/manifest.json
output/manifest.json
output/scf/            # local SCF cache used to create the exporters
```

The `.wfn` filenames follow the Multiwfn/atomwfn convention. For one-letter
element symbols, the filename contains a space before the suffix, for example:

```text
C .wfn
O .wfn
```

This is intentional and should not be renamed by cleanup scripts.
