# dyall-v4z references

Basis data for this bundle were obtained from Basis Set Exchange in NWChem
spherical format. The exact BSE API URL is recorded in `manifest.json`.

## Basis Set Exchange

Benjamin P. Pritchard, Doaa Altarawy, Brett Didier, Tara D. Gibson, and
Theresa L. Windus, "A New Basis Set Exchange: An Open, Up-to-date Resource for
the Molecular Sciences Community," J. Chem. Inf. Model. 2019, 59, 4814-4820.
DOI: 10.1021/acs.jcim.9b00725.

The BSE BSD-3-Clause notice is included in `../LICENSE-BSE-BSD-3-Clause.txt`.

## Upstream Dyall basis-set source

Kenneth G. Dyall, "Dyall dz, tz, and qz basis sets for relativistic electronic
structure calculations," Zenodo, Version 1, 2023. DOI: 10.5281/zenodo.7574629.
License: CC-BY-4.0.

The Zenodo source record lists the original Dyall basis-set references associated
with these relativistic basis families.
